flktran: a Filk File Translator

flktran.pl is a PERL script that translates ``.flk'' files to text or HTML. It can be found in ../TeX/.

It is worth noting that, since the original source files are in an extension of LaTeX, the conversion done by flktran is somewhat incomplete. This is most easily seen in such things as footnotes and mathematical expressions.
